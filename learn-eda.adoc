= 了解NetApp Workload Factory for EDA
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Workload Factory for EDA 可帮助您跨多个文件系统优化 FSx for ONTAP 。您可以自动化存储参数、分析性能限制，从而优化性能并降低运营成本，并获得 EDA 项目的见解。

它旨在与您的基础设施即代码 (IaC) 框架集成。

Workload Factory for EDA 提供仪表板和存储自动化框架及实用程序，帮助您管理多个 FSx for ONTAP文件系统。

它提供以下服务：

* link:task-configure-dashboard.html["EDA项目仪表盘"]：提供 FSx for ONTAP文件系统存储消耗的集中视图，以帮助您规划、评估成本并收集有关项目存储使用情况的信息。
* link:learn-eda-workloads.html["CI/CD"]：利用 FSx for ONTAP卷克隆功能，简化并缩短软件构建时间。
